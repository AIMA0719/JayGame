UI 스타일 방향 선택

운빨존많겜 / S2RD / 메이플 랜덤디펜스를 참고한 UI 디자인 톤 선택

레퍼런스 게임 UI 특징 분석

운빨존많겜

  • 어두운 배경 (다크 네이비/블랙)
  • 네온/글로우 액센트 (빨강, 노랑)
  • 둥근 카드형 UI
  • 하단 탭 네비게이션
  • 배틀 화면: 상단 웨이브 바 + 하단 소환/합성 버튼
  • 유닛 주사위 스타일 아이콘

S2RD

  • 깔끔한 다크 테마
  • 블루/시안 액센트
  • 미니멀한 아이콘
  • 그리드 기반 배틀 뷰
  • 심플한 HUD

메이플 랜덤디펜스

  • 원작 스타크래프트 맵 기반
  • 그리드에 유닛 배치
  • 합성(머지) 시스템
  • 웨이브 기반 진행
  • SP 기반 소환 경제
A

다크 네온 (운빨존많겜 스타일) - 추천

어두운 배경 + 강렬한 액센트 컬러. 게임 분위기와 가장 잘 어울리고, 유닛/이펙트가 돋보임.

B

다크 블루 (S2RD 스타일)

깔끔하고 차분한 블루 톤. 미니멀하고 모던한 느낌.

C

현재 중세 판타지 (유지)

현재 MedievalButton/WoodFrame 기반. 레퍼런스 게임들과 스타일이 많이 다름.
